# 📊 CodeAlpha - Data Visualization

## 📌 Description

This project demonstrates Data Visualization techniques using Python.

It converts raw data into meaningful visualizations using charts and graphs to identify trends, patterns, and insights for better decision-making.

This project was completed as part of the CodeAlpha Data Analytics Internship.

---

## 🚀 Features

- Bar Chart
- Line Chart
- Pie Chart
- Histogram
- Box Plot
- Correlation Heatmap

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
